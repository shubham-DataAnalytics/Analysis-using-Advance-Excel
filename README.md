# Analysis-using-Advance-Excel
This project focuses on analyzing a US healthcare dataset using Advanced Excel to identify trends, patterns, and key insights related to patient records, hospital performance, and healthcare costs. The objective was to transform raw healthcare data into meaningful information to support data-driven decision-making.

Key Work Performed:
Cleaned and prepared raw healthcare data by handling missing values, removing duplicates, and standardizing formats
Performed exploratory data analysis to identify trends in patient admissions, treatment costs, and hospital performance
Used Pivot Tables and Pivot Charts to summarize large datasets efficiently
Applied Excel formulas such as IF, VLOOKUP/XLOOKUP, COUNTIFS, SUMIFS, and conditional formatting
Created an interactive Excel dashboard to visualize KPIs and healthcare metrics
Generated insights to highlight cost drivers, high-volume hospitals, and patient distribution patterns
Tools & Skills:
Advanced Excel, Data Cleaning, Data Analysis, Pivot Tables, Dashboard Creation, Data Visualization
Outcome:
Converted complex healthcare data into clear reports and dashboards, improving understanding of operational and cost-related trends within the US healthcare system.
